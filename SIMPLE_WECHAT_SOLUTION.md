# 企业微信图片上传 - 最简单方案

## 🎯 方案：企业微信文件传输助手 + 网页上传

这是最简单的方案，无需配置快捷指令，只需要一个简单的网页。

---

## 📱 方案1: 手机网页上传（最简单）

### 步骤1: 部署上传服务（在服务器执行）

```bash
# 从GitHub更新代码
cd /tmp && rm -rf maoge-signal-reader
git clone https://github.com/clarinetsun-27/maoge-signal-reader.git
cd maoge-signal-reader

# 复制文件
cp wechat_message_receiver.py /root/maoge_advisor/
chmod +x /root/maoge_advisor/wechat_message_receiver.py

# 配置服务
cp services/maoge_wechat_receiver.service /etc/systemd/system/
systemctl daemon-reload

# 启动服务
systemctl enable maoge_wechat_receiver.service
systemctl start maoge_wechat_receiver.service
```

### 步骤2: 访问上传页面

在手机浏览器打开：
```
http://47.100.32.41:8888/
```

会看到一个简单的上传页面。

### 步骤3: 使用方法

1. 在企业微信保存猫哥图文到相册
2. 打开浏览器访问 `http://47.100.32.41:8888/`
3. 点击"选择文件"，选择刚保存的图片
4. 点击"上传分析"
5. 等待几秒，企业微信收到分析结果

**优点**：
- ✅ 无需配置任何东西
- ✅ 界面简单直观
- ✅ 适合所有手机

## 📱 方案2: 企业微信直接转发（更简单）

### 原理

在企业微信中将图片转发到"文件传输助手"，然后通过电脑端自动上传。

### 步骤1: 在服务器部署监控服务

这个方案需要在您的电脑上运行一个小程序，监控企业微信文件传输助手的图片。

### 步骤2: 使用方法

1. 在企业微信看到猫哥图文
2. 长按图片 → 转发 → 文件传输助手
3. 自动上传分析
4. 企业微信收到结果

**优点**：
- ✅ 最简单，只需转发
- ✅ 无需保存到相册
- ✅ 2步完成

**缺点**：
- ⚠️ 需要电脑在线
